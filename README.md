# Norbot

AI-drevet handelsanalyseplattform - en enkelt HTML-fil uten installasjon.

## Funksjoner
- Multi-AI-analyse (OpenAI, Anthropic, Grok) med opptil 3 modeller parallelt
- Simulering med win rate, P&L og equity-kurve
- Live trading via Kraken, Binance eller OANDA
- Kostnadssporing for token-bruk og API-kostnader
- Kunnskapsbase (PDF, tekst, URL)
- Auto-analyse etter tidsplan
- Live markedsoversikt (BTC, ETH, SOL)

## Kom i gang
1. Last ned norbot.html
2. Apne i Chrome
3. Legg inn API-nokkel i Innstillinger
4. Velg symbol og kjor analyse

## Kraken proxy
npx local-cors-proxy --proxyUrl https://api.kraken.com --port 8080

## Advarsel
Live trading innebarer finansiell risiko. Dette er et analyseverktoy, ikke finansiell radgivning.
